# Pencil Platform

Modern full-stack content publishing platform built with ASP.NET Core Web API and Angular, designed around scalable architecture, clean application boundaries, and real-world content management workflows.

## Overview

Pencil enables users to create, manage, and explore content through a structured publishing experience with authentication, personalized feeds, and interactive user flows.

## Tech Stack

### Backend

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* MediatR (CQRS Pattern)
* FluentValidation
* JWT Authentication

### Frontend

* Angular
* RxJS
* Reactive Forms
* Angular Material / Tailwind CSS

## Architecture & Engineering

* Clean Architecture implementation
* CQRS with MediatR for feature separation
* Layered backend structure (API, Application, Domain, Infrastructure)
* Reusable and modular frontend architecture
* Structured validation and exception handling
* Scalable service and repository design

## Core Features

* Secure authentication & authorization
* Post creation, editing, and management
* Personalized content feeds
* User profiles and interactions
* Responsive and modern UI
* RESTful API integration

## Goal

Pencil was built to simulate the architecture and workflows of modern publishing platforms while applying production-oriented backend and frontend engineering practices using .NET and Angular.
